# Portfolio

Personal portfolio website showcasing software engineering projects and technical skills.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Responsive design
- Multi-language support (FR/EN)

## Features

- Interactive project showcase
- Contact modal
- Background music with audio visualizer
- Intro loader animation
- Language switcher (French/English)

> **Note:** The contact form will be fully functional once the portfolio is hosted on Hostinger with a custom domain. Currently, it serves as a UI demonstration.

## Structure

portfolio/

├── about.html
├── src/
